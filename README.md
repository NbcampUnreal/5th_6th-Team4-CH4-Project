게임명 :	AFO (Arena Fighters Online)

장르	: 2 vs 2 멀티플레이어 팀 대전 액션

플랫폼	: PC (Dedicated Server 기반)


주요 특징	

* 5종의 고유 캐릭터 (팀당 중복 불가, 상대 팀 중복 가능)
* 5분 시간제한, 킬 스코어 방식 (가장 많은 킬을 달성한 팀 승리)
* 인게임 회복 버프 랜덤 스폰

네트워크	: 언리얼 엔진 Dedicated Server 기반 Full Replication

개발 기간	: 3주 (MVP: Minimum Viable Product 목표)

핵심 목표	: 기본적인 멀티플레이 접속, 팀/캐릭터 선택, 2vs2 대전 코어 루프 구현
